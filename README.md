## üöÄ Project Overview

This is a **cloud-based Escape Room web application** built using **Next.js (App Router)**.  
Users can:

- Play the Escape Room game  
- Submit their completion time  
- View the leaderboard  
- Edit and delete leaderboard entries  
- Generate dynamic result pages using a **serverless function (Lambda equivalent)**  
- Run automated Playwright tests  
- Run the app inside a **Docker container**  
- View **instrumented logs & performance metrics**
